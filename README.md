# 同人社团物語

一款同人创作经营模拟游戏。从18岁高考后的暑假开始，经历大学、工作、甚至全职同人的人生阶段，在热情与现实之间找到属于自己的创作之路。

## 玩法简介

游戏以**月**为单位推进。每个月你有有限的**时间**（天数），需要分配给不同的行动。你需要管理三项核心资源：

- **热情**：创作的根本动力。归零则游戏结束。工作压力、债务焦虑、摆烂都会消耗热情；完成作品、休息、好友支持可以恢复。
- **金钱**：印刷要钱，参展要路费，生活要开销。来源包括工资、卖本、接稿、打工。负债会加剧焦虑。
- **声誉**：在同人圈的知名度。出本、参展、宣传都能积累声誉，声誉越高销量越好、机会越多。

### 你可以做的事

| 行动 | 说明 |
|------|------|
| 创作同人本 | 多回合大项目（漫画、小说、绘本等），是积累声誉的主要途径 |
| 制作周边谷子 | 快速产出的小商品（亚克力、吧唧、色纸等），适合赚快钱 |
| 参加同人展 | 亲自参展或寄售，是卖出作品的主要渠道 |
| 接稿 | 用创作能力换钱，声誉越高收入越高 |
| 打工 | 稳定但微薄的收入，占用创作时间 |
| 宣传推广 | 社交媒体运营，扩大影响力 |
| 找搭子 | 寻找创作伙伴合作出本，但搭子有好有坏…… |
| 休息 | 恢复热情，有时候停下来也是一种选择 |
| 升级设备 | 永久提升作品质量、降低创作消耗 |
| 购买/出售谷子 | 用爱消费，或者断舍离变现 |

### 人生阶段

- **暑假**（高考后）：时间充裕，尽情创作的黄金期
- **大学**（4年）：课业压缩时间，但生活费有家里兜底
- **工作**：收入稳定但自由时间大幅减少，还会遭遇996、职业倦怠等职场事件
- **全职同人**（可选）：辞掉工作全身心投入创作，时间自由但没有固定收入，存款就是你的生命线
- **失业**（可能发生）：经济下行可能导致被裁，需要重新找工作

### 随机事件

游戏中会随机触发各种事件：经济衰退、996加班、灵感爆发、远程办公、好友援助……你无法完全控制命运，但可以选择如何应对。

## 结局

游戏共有 **7 种结局路线**，以及隐藏结局，每种路线会根据你的最终状态生成不同的叙事文本：

1. **热情燃尽**——最常见的结局。热情归零时，根据你的声誉、作品数、收入、人生阶段等生成专属的告别语
2. **商业出道**——声誉和作品积累到一定程度，出版社递来合约，从同人走向商业
3. **连续摆烂**——连续12个月有时间却不创作，热情在无所事事中悄悄熄灭
4. **债务压垮**——负债累累，困难家庭无力偿还，被迫放弃创作
5. **父母叫停**——学生时期欠债太多，父母勒令停止同人活动
6. **主动退出**——随时可以选择体面离场
7. **开放结局**——坚持到游戏结局，故事还在继续……

## 成就系统

游戏内置 **26 个成就**，涵盖创作里程碑、声誉突破、生存挑战、经济周期、情感韧性等多个维度。部分成就需要在极端条件下达成。

## 开始游戏

```bash
npm install
npm run dev
```

浏览器打开本地地址即可开始游戏。（注意浏览器本地不可用女神AI系统）
